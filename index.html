Problem

Find LCM of two Numbers. (Tiếng việt: tìm số nhỏ nhất chia hết cho cả 2 số nhập vào) => Nguồn: https://www.programiz.com.

Suggestions

+ Sử dụng vòng lặp for.

Nhắc lại kiến thức

Vòng lặp for 
                        
for(gia_tri_dau; bieu_thuc; bien_tang/giam) {
    khối code
}
+ Sử dụng kiến thức if. 
                        
if (điều kiện) {
    câu lệnh;
}

Code mẫu: 

                        
#include < stdio.h >
int main() {
     int n1, n2;
     printf("nhap n1: ");
     scanf("%d", &n1);
     printf("nhap n2: ");
     scanf("%d", &n2);
     int temp = 1;
     for (int i = 1; i <= n1 && i <= n2; i++) {
         if (n1 % i == 0 && n2 % i == 0 ) {
            if (i > temp) {
                temp = i;
            }
         }
     }
     int LCM = (n1 * n2) / temp;
     printf("LCM của 2 số nguyên %d, %d là %d", n1, n2, LCM);
    return 0;
}
